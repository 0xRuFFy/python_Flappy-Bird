
Flappy Bird clone in Python using Pyglet

Program call : python main.py [OPTION]

Options:
    ~ --help  | -h -> print this usage
    ~ --play  | -p -> start a playable game
    ~ --learn | -l [FILE] -> start maschine learing process write result to file (optional)
